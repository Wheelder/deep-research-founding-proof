# Wheelder — Innovation Proof & Project History

> A humble, transparent record of Wheelder's journey from a messy student project in 2020 to a focused AI research platform in 2026.

---

## A Note From the Developer

This repository exists because some people have questioned whether this work is original. Rather than argue, I'd prefer to simply share everything — the messy code, the naive implementations, the typos in the brand name, and every version in between — so anyone can see for themselves.

The code in these early archives is **rough, unpolished, and written by a student learning as they go**. It was never meant to be seen publicly. But for the sake of transparency and proving originality, everything is being shared openly. Anyone can download these files, inspect the code, and verify the timestamps independently.

This project has been a continuous learning journey. It started in **late 2020** as a simple experiment, continued through **distinct standalone projects**, went through a **major rebuild in 2022**, and has been actively developed since then. It began as a multi-app master platform and over time has evolved and narrowed its focus to what it is today.

**Wheelder is still a student project.** It is focused on research and learning, and has **not been commercialized**. It's still evolving.

## What's Inside These Archives

Anyone can download these files and inspect the code. Here's what you'll find:

- **`whl_project/`** folder — the original project directory name used since 2020
- **Naive, messy PHP code** — written by a student, not production-grade
- **Early AI integration** — OpenAI API calls using `gpt-3.5-turbo-16k-0613` (a model released June 2023 and deprecated June 2024)
- **"Digging Deeper" / "Deep Research"** — early implementations of progressive topic exploration, level by level
- **Dual-panel layout** — `#answerDiv` + `#imageDiv` side-by-side, the foundation of the current UI
- **Browser Text-to-Speech** — using `speechSynthesis` API before the Edge TTS upgrade
- **Keyword extraction** — pulling keywords from AI responses for further research
- **SQL database backups** — with Unix timestamps from October 2023 (`1697997845` = Oct 22, 2023)
- **"Wheeleder" branding** — the original misspelling, later corrected to "Wheelder"

## How to Verify (For Technical People)

### 1. Verify Google Drive Upload Dates

Google Drive API returns the `createdTime` (original upload date) and `md5Checksum` for any shared file. These are set by Google's servers and **cannot be modified** by the file owner.

```
GET https://www.googleapis.com/drive/v3/files/{FILE_ID}?fields=name,createdTime,modifiedTime,md5Checksum&key=YOUR_API_KEY
```

File IDs:
- `1sdiJ4zlSIr4APN8S8g7Btl08jghYiXZS` — wheeleder_files_sorted.zip
- `140MW9j7RRoT43rfeYmNmrYfe3TrZ2Lww` — wheeleder_latest_update_and_backup_04_02_2024.zip
- `1nUjcYIyA5kHy4En5ECDn9sVXxbYpITdi` — whl_project-Learn App renamed to ReLearnly...
- `1IbQMksPJ1rOtqFel1BDLomxMturf7IE0` — Relearn App-The Feature of Digging Deeper...
- `1M7Awxe7NGoMwNPs62EawOVrK65FahHhN` — ReLearnApp-double prompt entry...

### 2. Verify Git Commit History

```bash
# Clone and inspect the public repos
git clone https://github.com/Wheelder/relearn.git
git clone https://github.com/Wheelder/wheelder_platform.git

# View full commit history with dates
git log --format="%H %ai %s" --all
```

### 3. Verify Archive Integrity (OneDrive)

45 ZIP archives are stored on OneDrive with MD5 hashes. Verify any file:

```powershell
(Get-FileHash "filename.zip" -Algorithm MD5).Hash
```

The full hash manifest is available at [wheelder.com/timeline](https://wheelder.com/timeline).

### 4. Verify SQL Timestamps

Extract the SQL backup from any early archive and check the `created_at` fields:

```python
import datetime
print(datetime.datetime.utcfromtimestamp(1697997845))
# Output: 2023-10-22 18:24:05
```

### 5. Verify AI Model Temporal Evidence

The original code hardcodes `gpt-3.5-turbo-16k-0613`:
- Released: June 13, 2023
- Deprecated: June 13, 2024
- This proves the code was written during that window

The API key format (`sk-` prefix) further confirms pre-2024 origin — OpenAI switched to `sk-proj-` format in early 2024.
